# Instrucciones de uso de la práctica 4

Crear carpeta `full_dataset/` y subcarpeta `images/` dentro de `practica4/` e introducir el archivo csv y todas las imagenes dentro. Quedaría tal que así:

```bash
full_dataset/
├── ISIC_2019_Training_GroundTruth.csv
└── images/
    ├── image1.jpg
    ├── image2.jpg
    ...
```

El archivo `setup.ipynb` se encarga del resto.
